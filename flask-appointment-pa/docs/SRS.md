# Software Requirements Specification (SRS)

## Introduzione
Sistema di prenotazione appuntamenti per PA, sviluppato da N93403.

## Requisiti Funzionali
- RF-001: Il cittadino deve poter prenotare un appuntamento scegliendo ufficio, servizio, data/ora.
- RF-002: Il sistema deve prevenire doppie prenotazioni per lo stesso slot.
- RF-003: L’utente deve poter modificare o cancellare la prenotazione.

## Requisiti Non Funzionali
- RNF-001: Tempo di risposta < 2 secondi.
- RNF-002: Disponibilità ≥ 99,9% mensile.
- RNF-003: Conformità GDPR.
